# Comfort Serene Apartments 🏡

Comfort Serene is a modern residential property offering **well-designed 2-bedroom self-contained apartments** in **Elebu, Ibadan, Nigeria**. Each apartment is designed for comfort, privacy, and convenience, making it ideal for families, professionals, and individuals seeking quality accommodation.

---

## Features ✨

- 2 spacious bedrooms  
- 3 toilets and bathrooms (including en-suite)  
- Functional kitchen space  
- Comfortable living area  
- Fully self-contained layout  
- Peaceful environment for independent living  

---

## Tech Stack 

- **Frontend:** Next.js
- **Backend:** Node.js, Nextjs
- **Database:** MongoDB  
- **Hosting:** Vercel  
- **Image Storage:** Cloudinary (for property images)
